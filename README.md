Тема: Основи програмування на Python 
Мета роботи: Навчитись застосовувати основні конструкції мови Python, виконати всі приклади та з використанням AI створити власні приклади, які демонструють особливості кодових конструкцій Python.

Виконання роботи
Результати виконання завдань 1–8:
Типи даних: Розроблено код для ознайомлення з str, int, float, list, dict, tuple та set. Створено власний приклад профілю гравця "Vika", де використано список для інвентарю та словник для статистики.

Константи та зарезервовані слова: Виведено вбудовані константи (True, None) та повний список ключових слів Python за допомогою help("keywords").

Вбудовані функції: Програма вивела результати роботи функцій abs(), max(), sum() та len().

Цикли: Створено приклади циклу for для перебору списку букв та циклу while для імітації зворотного відліку (таймера).

Розгалуження: Реалізовано логіку перевірки здоров'я персонажа за допомогою if-elif-else та генерації випадкових чисел через randint.

Обробка помилок: Вивчено конструкцію try-except-finally. Програма успішно "відловила" помилку ділення на нуль та спробу звернення до неіснуючого ключа у словнику.

Контекст-менеджер: Створено та прочитано текстовий файл lab_report.txt за допомогою оператора with, що гарантує безпечне закриття файлу.

Лямбда-функції: Написано анонімну функцію для конвертації валюти (гривні в долари), що демонструє лаконічність синтаксису Python.
